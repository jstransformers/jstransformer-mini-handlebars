> `mini-handlebars` support for JS Transformers

## Install
```
npm i --save jstransformer-mini-handlebars
npm test
```


## Usage
> For more use-cases see the [tests](./test/index.js)

```js
var hbs = require('jstransformer')(require('jstransformer-mini-handlebars'));
```
